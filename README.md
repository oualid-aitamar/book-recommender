# book-recommender

Book Recommendation System

This Streamlit application allows users to search for books by keyword.  
It uses a CSV dataset of books and displays the results directly in a simple web interface.

# Features
- Search by title, author, or genre
- Simple Streamlit interface
- Displays results with basic information
